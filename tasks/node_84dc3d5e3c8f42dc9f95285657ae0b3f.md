# Add Frequently Used Prompts Feature

## Goal
Let users save prompts they use often and fire them with one click, reducing repetitive typing.

## Examples
- "What are my tasks for today?"
- "Summarise my notes from this week"
- "Add a journal entry"

## Requirements
- [ ] UI to create, name, and delete saved prompts
- [ ] Prompt picker accessible from the chat input (e.g. `/` shortcut or a button)
- [ ] Prompts stored persistently (local storage or brain config)
- [ ] Consider ordering by frequency of use

## Nice to Have
- Variable placeholders in prompts (e.g. `{date}`)
- Categories / tags on prompts

## Priority
Medium — quality-of-life improvement.
