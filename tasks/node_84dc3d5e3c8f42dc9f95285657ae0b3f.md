# Add Frequently Used Prompts Feature

## Goal
Let users save prompts they use often and fire them with one click, reducing repetitive typing.

## Examples
- "What are my tasks for today?"
- "Summarise my notes from this week"
- "Add a journal entry"

## Requirements
- [x] UI to create, name, and delete saved prompts
- [x] Prompt picker accessible from the chat input (e.g. `/` shortcut or a button)
- [x] Prompts stored persistently
- [x] Ordered by frequency of use

✅ **Done** — Frequently used prompts feature shipped.
