Currently automations only run while LMStudioClaw app is open.

**Goal:** Integrate with Windows Task Scheduler so that:
- A scheduled task can launch LMStudioClaw if it's not running
- Run the automation
- Optionally close again after

App should be controllable/launchable from Task Scheduler trigger. PC just needs to be on.
