# Tasks for Today — Dedicated Page

## Goal
Build a dedicated page that surfaces today's task nodes (and any journal entries tagged 'today') in a clean, focused view — no chat interface needed.

## Requirements
- [ ] Query brain for nodes of type `task` with `today` tag or a matching date
- [ ] Render tasks as a checklist
- [ ] Allow checking off / updating task status directly from the page
- [ ] Auto-refresh or manual refresh button
- [ ] Optionally show completed vs. pending tasks in separate sections

## Why This Is Valuable
Gives users a clean daily dashboard without needing to prompt the LLM — fast, glanceable, actionable.

## Priority
Medium-High — great standalone feature for daily workflow.
