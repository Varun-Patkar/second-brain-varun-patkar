## Task
Investigated adding [markitdown](https://github.com/microsoft/markitdown) to the Second Brain app to support ingesting rich file formats (HTML, PDF, DOCX, etc.).

**Status:** Abandoned on 2026-06-25.

### Reason for Abandonment
- **markitdown is a Python library** — it cannot be natively used in a Node.js project.
- Since Second Brain is built on Node.js, integrating markitdown would require a separate Python microservice or subprocess, which was deemed too complex/heavyweight for now.

### Potential Future Approach
- Could revisit with a Python sidecar service or explore a Node.js-native alternative for document-to-markdown conversion.
