## Windows Activity Time Tracker

**Idea:** Always-running Windows background app that tracks time spent on different activities and produces daily/weekly/monthly reports.

**Tracking scope:**
- YouTube — Windows app usage + phone usage via YouTube Data API
- Gaming — detect active game windows
- Work — detect work-related apps/windows

**Reports:** End-of-day summary, weekly digest, monthly overview. Show how much time spent on what.

**Key challenge:** Phone YouTube tracking — requires YouTube API integration (watch history or activity feed) since no direct OS hook.

**Status:** Idea stage as of 2026-06-25.
